# 浏览器起始页产品需求文档

## 1. 产品概述
一个仿终端风格的浏览器起始页，提供命令行交互界面，支持快速访问常用网站、搜索引擎和个性化配置。目标用户为追求高效、极客风格的浏览器用户。产品价值在于提供独特、高效的浏览器起始体验。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 核心权限 |
|------|----------|
| 用户 | 使用所有命令行功能、管理快速连接、自定义界面配置 |

### 2.2 功能模块
1. **主页面**：终端风格的命令行交互界面
2. **配置系统**：背景、光标等视觉元素的个性化设置

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 命令行输入区 | 显示提示符 `$username>`，接收用户输入的命令 |
| 主页面 | 输出显示区 | 显示命令执行结果和历史记录 |
| 主页面 | 快速连接列表 | 显示用户保存的常用网站链接 |
| 配置系统 | 背景设置 | 支持自定义背景图片或纯色背景 |
| 配置系统 | 光标设置 | 可配置光标颜色和样式（闪烁/静态） |

## 3. 核心流程

### 3.1 主要用户流程
1. 用户打开浏览器起始页，看到终端风格界面
2. 显示当前系统登录用户名作为命令提示符前缀
3. 用户输入命令与系统交互：
   - 使用 `use` 或 `list` 查看快速连接
   - 使用 `search 关键词` 进行搜索
   - 使用 `add URL` 添加新的快速连接
   - 使用 `config` 进入配置模式

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深色背景（#0a0a0a）+ 绿色荧光文字（#00ff00）+ 琥珀色高亮（#ffb000）
- **字体**：等宽字体（JetBrains Mono / Fira Code），营造真实终端感
- **布局风格**：全屏终端窗口，无边框沉浸式体验
- **视觉效果**：扫描线效果、微弱发光、CRT显示器质感
- **图标风格**：ASCII 字符或极简像素风图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 主页面 | 终端容器 | 全屏深色背景，可选背景图 |
| 主页面 | 提示符区域 | `$username>` 格式，绿色显示 |
| 主页面 | 输入框 | 等宽字体，闪烁光标 |
| 主页面 | 输出区域 | 滚动历史记录，命令结果展示 |
| 主页面 | 快速连接卡片 | 网站图标 + 名称 + URL |
| 配置面板 | 背景选择 | 图片上传/颜色选择器 |
| 配置面板 | 光标设置 | 颜色选择 + 样式切换 |

### 4.3 响应式设计
- Desktop-first 设计，优先桌面端体验
- 移动端自适应：调整字体大小和间距
- 触摸优化：移动端显示虚拟键盘友好的输入框

## 5. 功能规格说明

### 5.1 命令详细说明

#### use 命令
- **格式**：`use`
- **功能**：以交互式方式罗列所有快速连接，用户可通过编号或名称选择
- **输出示例**：
  ```
  [1] GitHub - https://github.com
  [2] Google - https://google.com
  选择要访问的链接（输入编号）:
  ```

#### search 命令
- **格式**：`search <关键词>`
- **功能**：根据网络环境自动选择搜索引擎
  - 检测到代理：使用谷歌搜索（https://www.google.com/search?q=关键词）
  - 未检测到代理：使用百度搜索（https://www.baidu.com/s?wd=关键词）
- **输出**：在新标签页打开搜索结果

#### add 命令
- **格式**：`add <URL>`
- **功能**：将网址添加到快速连接列表
- **验证**：检查 URL 格式有效性
- **输出示例**：
  ```
  ✓ 已添加: https://example.com
  ```

#### list 命令
- **格式**：`list`
- **功能**：列出所有已保存的快速连接
- **输出示例**：
  ```
  快速连接列表:
  [1] GitHub - https://github.com
  [2] Google - https://google.com
  共 2 个链接
  ```

#### config 命令
- **格式**：`config`
- **功能**：进入配置模式，可修改以下选项：
  - 背景图片（上传本地图片或输入图片 URL）
  - 背景颜色（当无背景图片时生效）
  - 光标颜色
  - 光标样式（闪烁/静态/下划线/方块）
- **存储**：使用 localStorage 持久化配置

### 5.2 特殊功能

#### 用户名显示
- 自动获取当前操作系统登录用户名
- Windows: 通过环境变量 `USERNAME` 获取
- macOS: 通过环境变量 `USER` 获取
- 显示在命令提示符中：`$username>`

#### 搜索引擎自动切换
- 检测方式：尝试访问 google.com，根据响应时间或状态判断是否可用代理
- 默认策略：优先谷歌，超时或失败则降级为百度

#### 数据持久化
- 快速连接列表存储在 localStorage
- 配置信息存储在 localStorage
- 数据格式：JSON

### 5.3 快捷键支持
- `Tab`：自动补全命令
- `↑/↓`：浏览历史命令
- `Enter`：执行命令
- `Ctrl+C` / `Esc`：清除当前输入
- `Ctrl+L`：清屏
